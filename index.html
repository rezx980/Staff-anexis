توثيق أوامر الإدارة — سيرفر أنكسيس ورلد

شرح مبسّط للأوامر: Warn, Timeout, Kick, Ban مع باراميترات وأمثلة قابلة للنسخ

النسخة: 2.0
⚠️

ممنوع الاستخدام إلا للبوتات المخصصة للإدارة

هذه الأوامر مخصصة حصريًا للبوتات المسؤولة عن إدارة السيرفر ولا يجوز استخدامها من قبل الأعضاء العاديين.

الأوامر الأساسية

⚠️ .warn

.warn <id> ?r "reason" تحذير

الوصف: إرسال تحذير لعضو معين (اختياري عرض السبب).

id
أيدي العضو (مطلوب).
?r
مفتاح اختياري للدلالة على أن النص التالي هو السبب.
reason
سبب التحذير (نصي، اختياري).
.warn 123456789012345678 ?r "مخالفة القوانين"

⏳ .t

.t <id> <duration> ?r "reason" ?dm true تايم آوت

الوصف: منع العضو من الدردشة لوقت محدد، مع إمكانية إرسال رسالة خاصة.

id
أيدي العضو (مطلوب).
duration
المدة بصيغة: 10m, 1h, 7d (مطلوب).
?r reason
السبب (اختياري).
?dm true
إذا أضفت هذا، يُرسل تنبيه في الخاص للعضو.
.t 123456789012345678 10m ?r "إزعاج" ?dm true

👢 .k

.k <id> ?r "reason" ?dm true طرد

الوصف: طرد العضو من السيرفر (يمكن دعوته مرة أخرى إذا سمح له).

id
أيدي العضو (مطلوب).
?r reason
سبب الطرد (اختياري).
?dm true
إرسال رسالة خاصة للعضو (اختياري).
.k 123456789012345678 ?r "إهانة الأعضاء" ?dm true

🔨 .b

.b <id> <duration> ?r "reason" ?dm true باند

الوصف: حظر العضو نهائياً أو مؤقتاً (بعض السيستمات تقبل مدة للباند).

id
أيدي العضو (مطلوب).
duration
مدة الباند (اختياري إذا النظام يدعم مؤقت). مثال: 7d أو permanent.
?r reason
السبب (اختياري).
?dm true
إعلام العضو بالخاص (اختياري).
.b 123456789012345678 7d ?r "تكرار السبام" ?dm true

الباراميترات (Parameters)

Id
هو الرقم الفريد للعضو على ديسكورد (مثل: 123456789012345678). يمكنك الحصول عليه عبر تفعيل وضع المطور ثم نسخ الـ ID.
Duration
تُستخدم لتحديد وقت مؤقت، الصيغة الشائعة: 10m (10 دقائق), 1h (ساعة), 7d (أيام).
Reason
السبب النصي الذي يظهر في السجل أو يُرسَل للعضو. جيد لتوثيق الإجراءات والشفافية.
?dm true
علم اختياري لإرسال رسالة خاصة قبل أو بعد تنفيذ الإجراء. مفيد للإعلام ولتجنب الشكوك.

نصائح وممارسات جيدة

  • دائماً وثّق السبب في السجل لتفادي النزاعات.
  • استخدم ?dm true لتقديم تحذير ودود قبل اتخاذ إجراء شديد.
  • تأكد من أن البوت يمتلك الصلاحيات اللازمة (Manage Messages, Moderate Members, Ban Members).
  • أوجد نظام استئناف أو مراجعة للأعضاء ليطلبوا تفسيرًا عند الضرورة.

تخصيص سيرفر أنكسيس ورلد

لجعل سيرفرك أكثر جاذبية واحترافية، إليك بعض النصائح بناءً على أفضل الممارسات [citation:2][citation:8]:

شعار السيرفر

صمم شعارًا فريدًا يعبر عن هوية سيرفر أنكسيس ورلد باستخدام أدوات مثل Canva [citation:6].

بانر ترحيبي

أضف بانرًا جذابًا بحجم 960x540 بكسل لتعزيز المظهر البصري للسيرفر [citation:9].

إيموجي مخصصة

أنشئ إيموجي وملصقات تعبر عن ثقافة السيرفر وشخصيته [citation:2].

أدوار مخصصة

صمم أدوارًا بألوان وأسماء تعكس اهتمامات الأعضاء وتنظيم السيرفر [citation:8].
